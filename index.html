Introducción de la materia



El lenguaje de maquetado HTML sirve como entorno gráfico al lenguaje de programación PHP, HTML contiene varios atributos o etiquetas donde puede ser ingresados los datos como en cualquier formulario, estos datos son procesado por el lenguaje PHP, aunque para ello es necesario la utilización de atributos de HTML, uno de los principales es el atributo name que se declara en las etiquetas de entrada (input) de HTML, en el atributo name se coloca el nombre de esa entrada de datos para que estos puedan ser llamados al archivo PHP y poder ser almacenado en una variable para su uso posterior. PHP contiene funciones con las cuales es mas fácil administrar el código o poder hacerlo mas corto, ya que estas funciones ya sean propias del lenguaje o las creadas por el programador de alguna manera facilitan el trabajo. Las funciones creadas por el programador pueden simplificar y reutilizar el código pudiendo usarlo en cualquier parte del script u otro script.
